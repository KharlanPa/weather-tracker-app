# WeatherTracker - Трекер погоды с избранными городами

## Описание проекта

**WeatherTracker** — мобильное приложение для отслеживания погоды в избранных городах. Приложение позволяет пользователю добавлять города в список избранных, просматривать текущую погоду и получать подробную информацию о погодных условиях.

## Основные функции

- Просмотр текущей погоды в выбранном городе
- Управление списком избранных городов (добавление/удаление)
- Поиск городов по названию
- Отображение подробной информации о погоде (температура, влажность, ветер, давление)
- Автоматическое обновление данных о погоде

## Экраны приложения

### 1. Главный экран (Home)
Отображает текущую погоду в выбранном городе:
- Название города
- Иконка погодных условий
- Текущая температура
- Описание погоды (облачно, солнечно и т.д.)
- Дополнительные параметры: влажность, скорость ветра, давление

### 2. Экран избранных городов (Favorites)
Список сохранённых городов с краткой информацией:
- Название города
- Текущая температура
- Иконка погоды
- Возможность удаления из избранного
- Кнопка добавления нового города

### 3. Экран поиска города (Search)
Поиск и добавление новых городов:
- Текстовое поле для ввода названия
- Результаты поиска
- Кнопка добавления в избранное

## Сценарии использования

### Сценарий 1: Просмотр погоды
1. Пользователь открывает приложение
2. На главном экране отображается погода в последнем выбранном городе
3. Пользователь видит температуру и погодные условия

### Сценарий 2: Добавление города в избранное
1. Пользователь переходит на экран избранных городов
2. Нажимает кнопку "+" для добавления
3. Вводит название города в поле поиска
4. Выбирает нужный город из результатов
5. Город добавляется в список избранных

### Сценарий 3: Переключение между городами
1. Пользователь открывает список избранных городов
2. Нажимает на нужный город
3. Главный экран обновляется с погодой выбранного города

### Сценарий 4: Удаление города из избранного
1. Пользователь открывает список избранных городов
2. Свайпает город влево или нажимает кнопку удаления
3. Город удаляется из списка

## Технические требования

### Навигация
- `BottomNavigationBar` с двумя вкладками: "Погода" и "Избранное"
- Переход на экран поиска через кнопку "+"

### Хранение данных
- **Локальное хранилище**: `shared_preferences` или `sqflite` для сохранения списка избранных городов
- **Кэширование**: временное хранение последних данных о погоде

### API
- **OpenWeatherMap API** для получения данных о погоде
- Эндпоинты:
  - Текущая погода: `/weather?q={city}`
  - Поиск города: `/geo/1.0/direct?q={city}`
